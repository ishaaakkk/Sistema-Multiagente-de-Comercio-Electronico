Description

Ontología del sistema multiagente de comercio electrónico. Define los conceptos compartidos por los agentes para representar productos, ofertas, pedidos, pagos, envíos, devoluciones, valoraciones y recomendaciones. El dominio se modela con vocabulario propio para evitar dependencias innecesarias de ontologías externas; se mantiene time para instantes temporales.
